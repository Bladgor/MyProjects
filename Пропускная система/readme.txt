На линуксе надо помимо библиотек из requirements.txt надо ещё установить библиотеку tkinter
командой:
sudo apt install -y python3-tk

Я использовал эту, т.к. внушает больше доверия и ставится вне виртуального окружения,
но потом уже можно использовать в любом виртуальном окружении.

Есть ещё одна:
apt-get install python3-tk
Но её я не пробовал)
